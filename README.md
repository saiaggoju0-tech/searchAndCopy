# 🚛 SSP Truck Dashboard

A **prototype truck operations dashboard** developed to demonstrate an idea for reducing manual effort in truck tracking workflows.

This prototype shows how truck data can be **filtered by Origin/Lane and Critical Pull Time (CPT)**, how **non-completed trucks** can be identified quickly, how **53ft and 26ft equipment** can be counted automatically, and how only the **required visible active truck data** can be copied with a single click.

---

## 📌 Problem Solved

In relay operations, manually checking truck statuses such as:

* Scheduled
* Loading in Progress
* Trailer Attached

for a specific **site and Critical Pull Time (CPT) time** takes time and increases the chance of missing trucks.

This dashboard automates that process by:

* Instantly filtering required rows
* Counting available trucks
* Separating completed vs active trucks
* Copying only visible non-completed truck data

This reduces **manual effort, reporting time, and operational errors**.

## 🚀 How It Works

1. Take the **Origin/Lane** and **Critical Pull Time (CPT)** from the FMC page
2. Enter **Origin/Lane** in the search box
3. Enter **Critical Pull Time (CPT)**
4. Dashboard filters matching trucks
5. Counts active **53ft and 26ft trucks**
6. Click **Copy Button**
7. Only visible **non-completed trucks** get copied automatically
